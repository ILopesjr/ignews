# Projeto ignews

Tecnologias usadas:

• NextJs, TypeScript, SASS, Stripe API, Primic CMS e FaunaDB.

Projeto de aprendizagem nas tecnológias mencionadas.

O projeto tem como finalidade simular assinatura de conteúdo de um blog.
